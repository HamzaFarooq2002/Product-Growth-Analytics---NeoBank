# Product-Growth-Analytics - NeoBank
Growth Analytics for a Neobank
This project simulates the work of a Growth Analyst at a digital banking startup. 
The goal is to analyze customer activity, transaction data, and marketing campaigns to extract insights that drive product growth and marketing efficiency.

The project is built using SQL-based analytics workflows, replicating real-world fintech analytics tasks.

Project Objective:

Digital banks rely heavily on data-driven growth strategies. This project demonstrates how analytics teams evaluate:

• Customer engagement
• Product adoption
• Revenue generation
• Marketing campaign performance
• Customer retention and churn risk

The dataset simulates operations for a neobank offering multiple financial products.
Tech Stack: (MYSQL)
